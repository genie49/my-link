# 마이링크 (MyLink)

> 흩어진 모든 링크를, 하나의 주소로.

인스타그램·유튜브·블로그·포트폴리오 등 여러 링크를 프로필 하나에 모아 공유하는 **링크 인 바이오(Link in Bio)** 서비스의 랜딩 페이지입니다. AI 코딩 실습으로 만든 나만의 링크트리 프로젝트입니다.

## ✨ 주요 기능 (랜딩 페이지)

- **에디토리얼 디자인** — 따뜻한 크림 배경 + 토마토 레드 단일 액센트, Fraunces / Noto Serif KR 타이포그래피
- **인터랙티브 링크 프리뷰** — 데모 프로필 카드에서 링크를 눌러보며 클릭 수가 올라가는 모습 확인
- **반응형 레이아웃** — 모바일·데스크톱 모두 대응
- **부드러운 진입 애니메이션** — staggered reveal, 마키, 종이 질감 텍스처

## 🛠 기술 스택

| 구분 | 사용 도구 |
| --- | --- |
| 프레임워크 | Next.js 16 (App Router) |
| 언어 | TypeScript |
| 스타일 | Tailwind CSS v4 |
| 폰트 | Fraunces · Noto Sans KR · Noto Serif KR |
| 배포 | Vercel |

## 🚀 로컬 실행

```bash
npm install
npm run dev
# http://localhost:3000
```

## 📦 빌드

```bash
npm run build
npm start
```
