# my-link

AI 코딩 실습 — **마이링크(MyLink)** 프로젝트 저장소입니다.

흩어진 여러 링크(인스타그램·유튜브·블로그·포트폴리오 등)를 프로필 하나에 모아 공유하는 링크 인 바이오 서비스를 만듭니다.

## 📁 폴더 구조

```
my-link/
└── mylink/        ← 마이링크 전용 폴더 (Next.js 앱)
    ├── app/       랜딩 페이지 · 컴포넌트
    ├── public/
    └── README.md  프로젝트 상세 문서
```

➡️ 마이링크 폴더: [`/mylink`](./mylink)

## 🌐 라이브 주소

Vercel로 배포되어 누구나 접속할 수 있습니다.

- **MyLink:** _(배포 후 업데이트)_

## 🛠 기술 스택

Next.js 16 · TypeScript · Tailwind CSS v4 · Vercel
